# 泰八林業 LP / Next.js + Tailwind CSS

添付デザインをもとに、LP本体を **HTML / JSX / Tailwind CSS のコンポーネント** として組み直した実装です。
前回のような「LP全体を画像として貼る」方式ではありません。

## 構成

- `app/page.tsx`：LPページ本体
- `components/TaihachiLanding.tsx`：セクション単位のコンポーネント
- `app/globals.css`：紙質背景、カード、見出し、フロー線などの共通スタイル
- `public/assets/illustrations/`：手描き風イラスト・写真風素材・ロゴマークなどのPNG素材

## UIの作り方

HTML / Tailwind CSSで組んでいるもの：

- ヘッダー
- ヒーロー見出し、CTAボタン
- お悩みカード
- サービスカード
- プロセス図
- 対応エリア
- お客様の声カード
- 施工事例グリッド
- FAQ
- フッターCTA

画像として使っているもの：

- 手描き風の人物・木・山・丸太などの装飾素材
- カード内の小さなイラスト
- 施工事例風の写真素材
- QR風素材

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで以下を開いてください。

```bash
http://localhost:3000
```

## レスポンシブ

- PC：横並びグリッド、ヒーロー左右分割
- タブレット：2〜3列グリッドへ縮小
- スマホ：1〜2列、CTAは縦積み
